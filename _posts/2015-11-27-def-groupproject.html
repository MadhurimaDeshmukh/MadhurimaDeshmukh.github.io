Week 9 is done. I am three quarters through General Assembly Seattle now. The project I worked on for the past week has been Fare Start. It was a group project built in Ruby on Rails to get Uber and Lyft quotes before picking which one to use.

We wanted to build something that people would use while still meeting the project requirements. The requirements were to create an application using at least two database models, user authentication, and CRUD functionality. One of the models saved data from every search to generate a map using D3.js. The map can be sorted by total searches per state, average Uber fare, and average Lyft fare. Another feature we added was mobile deep linking where the user can open the Uber or Lyft app from the search results page with the to and from already filled in for them. We also used four APIs to build Fare Start: Uber, Google Maps Javascript, Google Maps Geocoding, and Freegeoip. Lyft did not have an available API at the time.

This project was the first time I worked in a group and it was a learning experience. We had to assign different roles to each member instead of just going for it yourself. We also were working on one repository, so we had to quickly learn how to avoid merge conflicts on GitHub. I also got to work with Sass, which made changing colors much quicker.
