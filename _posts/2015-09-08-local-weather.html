I had a busy Labor Day weekend. After finishing the random quote generator I jumped straight into another project on FreeCodeCamp. This one was to show the visitor’s local weather in Fahrenheit or Celsius and to have a background picture similar to what it looked like outside.

I used the same Bootsrap template that I used for the random quote generator because it was another project for developing my JavaScript and jQuery skills.

The project required the use of two APIs. I used an API to get the IP address of the visitor to find out where they were from. I also used an API to get the local weather of the visitor based on the latitude and longitude of the IP address. The temparture was easy to get and convert from Kelvin to Fahrenheit or Celsius. The weather proved a little more difficult because I could not find a list of all the possible outputs. I had to use the weather icon output, which has two possibilities for day and night so that is why I used a substring of only the first two characters. Whatever the two character string comes back as is what the background image will be. It is not as accurate as the main weather output, but it still works for what I am doing.

I tried using a switch statement instead of an if... else statement, but I could not get it to work for this project. I could improve on the code by figuring out if a switch statement is possible to use.

You can view the project on CodePen and get all of the code on GitHub.
