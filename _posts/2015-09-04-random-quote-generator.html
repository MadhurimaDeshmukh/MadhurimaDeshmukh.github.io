I am learning more with FreeCodeCamp on my own than I thought was possible. I am very happy with my progress leading up to the General Assembly Seattle web development immersive program that I start later this month. I completed a basic front end development project yesterday, but I am still making a few tweaks here and there based on feedback and new ideas.

The project was to make a random quote generator that the visitor could see new quotes and tweet one if they would like. I decided to make one with Star Wars quotes because I just started watching the movies this week and thought it would be fun to do with the new movie coming out soon.

I have been using Bootstrap for all of my front end development projects lately. I love how easy it is to use and how great the pages look using it. But the main point of this project was to get some JavaScript and jQuery practice.

One quote is picked at random from an array of quotes in the function. The quote is checked for length since 140 characters is the maximum for a tweet and a hashtag, emoji, and link are being added. C-3PO has an emoji available that I wanted to use, so those quotes have to be checked separately. If the quote is small enough then it is being added to the HTML classes. New variables are also being set to use in the jQuery for the visitor to be able to send a tweet.

In the jQuery the tweet button is initially hid. Originally the visitor was able to tweet the landing page quote, but I wanted them to generate a new quote. Once the visitor clicks to generate a new quote the tweet button will show again. The tweet button will also have a new link attributed to it that will generate a tweet for the quote currently on the page.

You can view the project on CodePen and get all of the code on GitHub.
